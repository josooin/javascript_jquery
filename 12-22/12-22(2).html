02자바스크립트 내장객체_배열(array)객체

배열(array)객체란?
연관된 데이터를 하나처럼 관리하고 싶을 때 사용
복수의 데이터를 제어할 수 있는 다양한 함수를 제공한다.
배열의 함수를 이용한 데이터의 편리한 조작이 가능하다.
연관된 데이터
과일 : 수박, 참외, 포도, 딸기, 복수아, 사과, 귤, 바나나, 자두, 오렌지..........
색상 : 파랑, 빨강, 노랑, 검정, 흰색......
개인정보 : 이름, 나이, 주소, 주민번호, 전화번호, 아이디, 비밀번호, 취미, 특기, 성별......
쇼핑정보 : 상품명, 가격, 할인률, 상품코드, 주문자, 판매자......
예약정보 : 방이름, 호수, 날짜, 가격, 서비스, 인원수.......

1. 배열 객체 선언하기

방법1. let array = new Array(data1, data2, data3, data4........ dataN);

방법2. let array = [data1, data2, data3, data4......dataN];

예시1) let fruits = new Array('수박', '참외', '귤', '감', '사과', '포토', '복숭아'....)

예시2) let fruits = ['수박', '참외', '귤', '감', '사과', '포토', '복숭아'....]

2. 배열 객체 호출하기

배열객체의 데이터를 호출하고자 할 때는 인덱스 번호를 사용하는데 번호는 0부터 시작한다.

실습예제. 중화요리 음식 이미지를 사용하여 요일별 음식추천하기

날짜객체를 사용하여 현재 요일 번호를 구해서 배열번호에 맞는 이미지 파일명을 찾아 img태그에 대입해서 음식사진이 요일별로 출력되게 한다.
